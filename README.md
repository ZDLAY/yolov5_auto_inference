## 使用教程
打开`main.py`第`94`行，修改你的conda虚拟环境名称

需要提前安装好Yolov5的运行环境

直接运行`main.py`选择你的路径点击`开始自动标注`按钮即可一键**自动标注**

如果没有请直接使用python运行`auto_inference.py`
